Welcome to the Installation Guide


Before you can install and run the project, you need to install XAMPP on your computer.

Follow the steps below to install XAMPP:

  1. Download XAMPP from the following link: Download XAMPP
  2. Run the installer and follow the on-screen instructions to complete the installation.
  3. Make sure to install XAMPP in the default installation directory C:/Xampp.

Once you have installed XAMPP, you can proceed with downloading the project. Follow the steps below:

  1. Click on the following link to download the project: Download the project
  2. After the download is complete, extract the project files to a location of your choice.

To install the project, follow the instructions below:

  1. Navigate to the extracted project folder.
  2. Locate the setup.exe file and double-click on it to start the installation process.
  3. Follow the on-screen instructions to complete the installation. Make sure to select the default installation directory C:/Xampp/htdocs.

Once the installation is complete, you can access the project in two different ways:

  • Desktop: On your desktop, you will find a shortcut to the Ceres application. Double-click on this shortcut to launch the project.
  • Windows Menu: In the Windows Start menu, search for the "Ceres" shortcut in the list of installed programs and click on the Ceres application icon to launch the project.

Once you have launched the project, it will be accessible in your web browser

That's it! You can now start using the project!
