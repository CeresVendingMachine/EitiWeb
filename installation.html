CERES

Avant de pouvoir installer et exécuter le projet, vous devez installer XAMPP sur votre ordinateur.

Suivez les étapes ci-dessous pour installer XAMPP :

  1. Téléchargez XAMPP à partir du lien suivant : Télécharger XAMPP
  2. Exécutez l'installateur et suivez les instructions à l'écran pour terminer l'installation.
  3. Assurez-vous d'installer XAMPP dans le répertoire d'installation par défaut C:/Xampp.

Une fois que vous avez installé XAMPP, vous pouvez procéder au téléchargement du projet. Suivez les étapes ci-dessous :

  1. Cliquez sur le lien ci-dessus pour commencer le téléchargement du projet : Télécharger le projet
  2. Une fois le téléchargement terminé, extrayez le contenu du projet dans un emplacement de votre choix.

Pour installer le projet, suivez les instructions ci-dessous :

  1. Naviguez jusqu'au dossier du projet extrait.
  2. Localisez le fichier setup.exe et double-cliquez dessus pour démarrer le processus d'installation.
  3. Suivez les instructions à l'écran pour terminer l'installation. Assurez-vous de sélectionner le répertoire d'installation par défaut C:/Xampp/htdocs.

Une fois l'installation terminée, vous pouvez accéder au projet de deux manières différentes :

  • Bureau : Sur votre bureau, vous trouverez un raccourci vers l'application Ceres. Double-cliquez sur ce raccourci pour lancer le projet.
  • Menu de Windows : Dans le menu de démarrage de Windows, recherchez le raccourci "Ceres" et cliquez sur l'icône de l'application Ceres pour lancer le projet.

Une fois que vous avez lancé le projet, il sera accessible dans votre navigateur web.

Voilà, vous pouvez maintenant commencer à utiliser le projet !
